2. Login Page

Full-viewport hero with luxury architectural photography background (Unsplash). Oversized serif headline, glassmorphism login card floating center-right. Subtle teal glow on input focus. Ghost CTA button.

3. Dashboard Layout

Sidebar collapses to hamburger on mobile. Top nav bar with thin teal bottom border. Page titles set in 48px Playfair Display with tight line-height. Content cards have generous padding (40px), thin teal top border, and soft drop-shadow.

4. Section Variations

Products

Asymmetric 3-column grid. Large hero product card spans 2 cols. Teal-tinted card borders. Quick-action FAB in deep teal.

Transactions

Clean table-first layout with teal-striped rows. Stat cards at top use glassmorphism. Date filters stacked horizontally with teal pill-style active state.

Staff

Card grid with circular avatar insets. Each card has a thin left teal border accent. Role badges in soft teal capsules.

5. Components

  • Buttons: Outlined teal with 4px radius, uppercase DM Sans, letter-spacing .1em. Hover: teal fill, white text.
  • Inputs: Transparent background, teal bottom border-only. Focus: soft teal glow.
  • Cards: White surface, 40px padding, 12px radius, thin teal top accent stripe, box-shadow: 0 8px 30px rgba(40,81,86,.08).
  • Tables: Minimalist, no vertical lines, teal-tinted alternate rows, teal header underline.
